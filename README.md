# HerdSet7K
This repository provides a sample preview of the HerdSet7K dataset, which is used in our manuscript for UAV-based dense cattle counting and localization. The dataset consists of high-resolution aerial images captured over diverse farm environments, covering variations in illumination, scale, occlusion, and background clutter.
The full dataset, including all images, point annotations, train/val/test splits, will be made publicly available in this repository upon official acceptance of the manuscript. We are committed to releasing the complete dataset to facilitate reproducible research and further development in UAV-based livestock monitoring.
